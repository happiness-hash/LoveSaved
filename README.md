# 心意 Web

一个移动端优先的轻量 Web 应用原型，用来承载双人心意记录、兑换申请、消息提醒和共享吻账户。

## 打开方式

这是纯静态项目，不依赖构建工具。

1. 直接双击根目录的 `index.html`
2. 或者用任意静态文件服务器打开当前目录

## 当前能力

- 首页总览：我的心意、TA 的心意、我们的吻、最近动态
- 心动页：单条完整记录浏览，顶部按钮新增
- 兑换页：筛选、发起申请、审批
- 消息页：提醒和待处理
- 我们的吻：增减与文字记录

## 说明

- 当前数据保存在前端内存里，刷新页面会恢复到初始假数据
- 图片和语音上传还没接真实服务，先用链接和时长占位
- 项目已经移除微信小程序和云开发结构
